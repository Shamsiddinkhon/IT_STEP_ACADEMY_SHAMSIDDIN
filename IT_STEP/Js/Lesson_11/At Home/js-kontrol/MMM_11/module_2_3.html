Задание 3 Реализовать класс, описывающий новостную ленту. Класс должен содержать: ■■ массив новостей; ■■ get-свойство, которое возвращает количество новостей; ■■ метод для вывода на экран всех новостей; ■■ метод для добавления новости; ■■ метод для удаления новости; ■■ метод для сортировки новостей по дате (от последних но- востей до старых); ■■ метод для поиска новостей по тегу (возвращает массив новостей, в которых указан переданный в метод тег). Продемонстрировать работу написанных методов.
