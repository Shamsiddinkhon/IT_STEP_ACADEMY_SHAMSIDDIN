Чтобы запретить пользователю менять собственноручно размер Блока textarea нужно написать resize: none; Высоту и ширину также можно менять с помощью css. 

Для того чтобы убрать дефолтные(стили по умолчанию) у кнопок и полей ввода
нужно прописать свойства outline:none и border: none;

Также для кнопок, чекбоксов (кнопки с галочкой), радиокнопок убрать дефолтные стили нужно написать для них 
appearance: none;
-webkit-appearance: none;

Второй вариант - это прописывание для поля display:none, но тогда нужно создать свой чекбокс с помощью псевдоэлемента ::before и ::after от тега label

Для создания кастомного скролла можно использовать 

::-webkit-scrollbar{   - управление самим блоком скролла
   
}
::-webkit-scrollbar-thumb{ - управление самим ползунком внутри блока скролла
}
Иконки в поля ввода добавляются с помощью позизионирования absolute.
Но это только один из вариантов.
